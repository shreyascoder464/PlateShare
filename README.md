# 🍽️ PlateShare

PlateShare is a community-driven food sharing platform developed to reduce food wastage by connecting food donors with individuals and organizations in need. The application enables users to donate surplus food, browse available donations, and manage food requests through an intuitive web interface.

---

## Features

* User Registration & Login
* Food Donation Management
* Browse Available Food Listings
* Food Request System
* User Dashboard
* Responsive User Interface
* Backend API Integration

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
npm install
```

### Run Backend

```bash
cd backend
npm install
npm start
```

### Run Frontend

```bash
cd frontend
npm install
npm start
```
